GET /filters returns

```
currencies
languages

filter_groups
    filters
        id
        name
        type (radio, integral, activity_facility)
        {
            default_options {id, name}
            number
            icon
        }
```

GET /dorms accepts

```
currency=USD
language=EN

filters
    id
    {
        choosen_options_ids
        min_value max_value
        // activity facility doesn't need an extra field
    }
```

GET /dorms returns

```
dorms
    id
    cover

    stars
    number_of_reviews

    geo_longitude
    geo_latitude
    address

    facilities
        name
        icon (could be empty)
    
    activities
        name
        icon (could be empty)

    number_of_found_rooms
    rooms
        cover
        price
        room_type
        people_allowed_number

        facilities
            name
            icon (could be empty)
```

GET /dorms/{id}/reviews

```
reviews
    student_name
    stars
    description
```
