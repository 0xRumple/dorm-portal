POST /dorm/{id} accepts
```
currency=USD
language=EN

filters
    id
    {
        choosen_options_ids
        min_value max_value
        // feature doesn't need an extra field
    }
```

GET /dorms/{id} returns
```
name

cover

photos
    url
    is_3d

geo_longitude
geo_latitude
address

about

contact_email
contact_number

dorm_features
    name
    icon (optional field)

number_of_reviews
reviews_average
reviews
    student_name
    stars
    description

rooms
    id
    rooms_left
    photos [cover, rest photos] // just images src
    room_type
    price
    people_allowed_number

    features
        name
        icon (optional field)
```
