```
name
email
password
```

```
email
```

```
```

POST /auth/resend-confirmation/ returns 400 Bad request
```

```

POST /auth/login accepts
```
email
password
```

POST /auth/login returns 200 OK
```
name
is_manager
current_step (email_confirmed=2, payment_uploaded=3)
reservation_id (null = doesnt have a reservation yet)
```

POST /auth/login returns 403 Forbidden
```
please confirm your email
```

POST /auth/login returns 400 Bad request
```
you entered wrong email or password
```

POST /auth/confirm-email returns 200 OK
```
token
```

POST /auth/confirm-email returns 400 Bad request
```
token expired, try sending another email
```
