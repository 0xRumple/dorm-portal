
GET /locale returns
```
currencies
    symbol
    code

languages
    name
    code
```

GET /filters returns
```
main_filters
    id
    name
    (checkbox true) (integral false)
        options {id, name}

additional_filters
    id
    name
    (checkbox true) (integral false)
        options {id, name}
    (checkbox false) (integral true)
        min_value max_value value [min_value,max_value]

dorm_features
    id
    name

room_features
    id
    name
```

POST /dorms accepts
```
currency=USD
language=EN

main_filters
    id
    {
        choosen_options_ids
        min_value max_value
    }

additional_filters
    id
    {
        choosen_options_ids
        min_value max_value
    }

dorm_features
    id

room_features
    id
```

```
dorms
    id
    cover

    stars
    number_of_reviews

    geo_longitude
    geo_latitude
    address

    features
        name
        icon (optional field)

    rooms_left_in_dorm
    rooms

        room_type
        people_allowed_number
```
