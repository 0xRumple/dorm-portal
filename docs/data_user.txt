POST /user/register accepts
```
name
email
password
```

POST /user/resend-confirmation/ accepts
```
email
```

POST /user/resend-confirmation/ returns 200 OK
```
role (manager || student)
```

POST /user/resend-confirmation/ returns 400 Bad request
```
user not registered
```

POST /user/login accepts
```
email
password
```

POST /user/login returns 200 OK
```
current_step (email_confirmed=2, payment_uploaded=3)
reservation_id (null = doesnt have a reservation yet)
is_manager
```

POST /user/login returns 403 Forbidden
```
please confirm your email
```

POST /user/login returns 400 Bad request
```
you entered wrong email or password
```

POST /user/confirm-email/{token} returns 200 OK
```
```

POST /user/confirm-email/{token} returns 400 Bad request
```
token expired, try sending another email
```
