
GET /locale returns
```
currencies
    symbol
    code

languages
    name
    code
```

GET /filters returns
```
filter
    id
    name
    (checkbox true) (integral false)
        options {id, name}
    (checkbox false) (integral true)
        min_value max_value value [min_value,max_value]
```

POST /dorms accepts
```
currency=USD
language=EN

filters
    id
    {
        choosen_options_ids
        min_value max_value
        // activity facility doesn't need an extra field
    }
```

POST /dorms returns
```
dorms
    id
    cover

    stars
    number_of_reviews

    geo_longitude
    geo_latitude
    address

    facilities
        name
        icon (optional field)
    
    activities
        name
        icon (optional field)

    number_of_found_rooms
    rooms
        id
        cover
        price
        room_type
        people_allowed_number

        facilities
            name
            icon (optional field)
```

GET /dorms/{id}/reviews
```
reviews
    student_name
    stars
    description
```
